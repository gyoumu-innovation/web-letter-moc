# 機能設計書：ユーザー情報確認機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-004 |
| 機能名 | ユーザー情報確認機能 |
| 対象ユーザー | 顧客利用者（ログイン済み） |
| 対応画面 | ユーザー情報画面（profile.html） |

---

## 機能説明

ログインユーザーの会社情報・連絡先情報を閲覧する（編集は社内担当者が行う）。

## 表示項目

| 項目名 | 表示内容 |
|---|---|
| 会社名 | companies.name |
| 得意先ID | companies.customer_code |
| メールアドレス | users.email |
| 電話番号 | users.phone |
| 担当者名 | users.name |

## 処理フロー

```
1. ヘッダーのユーザーメニュー「ユーザー情報」から遷移
2. ログインユーザーのusers・companies情報を取得して表示
```
