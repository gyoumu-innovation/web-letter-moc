# 機能設計書：送付リストテンプレートダウンロード機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-013 |
| 機能名 | 送付リストテンプレートダウンロード機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 新規注文作成画面 Step5（文面・送付先） |

---

## 機能説明

送付先リストを入力するためのExcelテンプレートファイルをダウンロードする。

## 処理フロー

```
1. Step5の「テンプレートをダウンロード」ボタンをクリック
2. ストレージから固定のテンプレートファイルを取得（IF-03）
3. ブラウザにダウンロード
```

## テンプレートファイル仕様

| 項目 | 内容 |
|---|---|
| ファイル形式 | Excel（.xlsx） |
| ファイル名 | mailing_list_template.xlsx |
| 必須列 | 氏名、郵便番号、都道府県、住所1 |
| 任意列 | 住所2、建物名、宛名敬称 |

## 外部IF

- IF-03（送付リストテンプレートダウンロード）を利用
