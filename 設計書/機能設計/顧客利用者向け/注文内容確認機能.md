# 機能設計書：注文内容確認機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-007 |
| 機能名 | 注文内容確認機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 注文詳細画面 Step7（入稿フォーム確認ステップ） |

---

## 機能説明

注文確定前に全ステップの入力内容を一覧で確認する（入稿フォームStep7）。  
内容に問題があれば各ステップへ戻って修正できる。  
※ステータスが「確認待ち」の場合は注文詳細画面（注文ステータス確認機能）で確認する。

---

## 表示項目

入稿フォームStep7（お申し込み）で以下を表示：

| セクション | 表示内容 |
|---|---|
| 封筒 | Step1の入力内容（サイズ・色・送付方法・オプション） |
| 便箋 | Step2の入力内容（書き方・フォント・オプション） |
| 差出人情報 | Step3で選択した差出人 |
| 発送日・通数 | Step4の入力内容 |
| 文面 | Step5の入力内容（文面テキスト・送付リスト件数） |
| 同梱物 | Step6の入力内容 |
| 料金（概算） | 通数×単価の概算 |

## 処理フロー

```
1. Step7に到達した時点で全ステップの入力内容を表示
2. 「修正する」リンクをクリック → 対象ステップへ戻る
3. 「注文する」ボタンクリック → 注文機能（F-U-009）へ委譲
```
