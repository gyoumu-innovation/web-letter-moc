# 機能設計書：文面プレビュー機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-015 |
| 機能名 | 文面プレビュー機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 新規注文作成画面 Step5（文面・送付先） |

---

## 機能説明

入力した文面テキストをStep2で選択したフォント・書き方（縦/横）でプレビュー表示する。

## 処理フロー

```
1. Step5の「プレビュー」ボタンをクリック
2. 入力中の文面テキスト・選択フォント・書き方をモーダルで表示
3. モーダルを閉じて編集を継続
```

## 表示仕様

| 項目 | 内容 |
|---|---|
| 表示形式 | 便箋レイアウトのモーダル |
| フォント | Step2で選択したフォント |
| 書き方 | 縦書き or 横書き |
| 宛名 | 「宛名を追加する」がONの場合、先頭に「{宛名}様」を表示 |

## 備考

- プレビューはクライアントサイドで処理（DBアクセスなし）
- フォントのレンダリングはCSSで再現
