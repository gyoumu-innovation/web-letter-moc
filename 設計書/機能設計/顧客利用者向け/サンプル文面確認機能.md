# 機能設計書：サンプル文面確認機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-016 |
| 機能名 | サンプル文面確認機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 新規注文作成画面 Step2（便箋）・Step5（文面） |

---

## 機能説明

社内担当者がアップロードしたサンプル文面（フォントや書き方のサンプル）を確認する。

## 処理フロー

```
1. Step2のフォント選択カードのズームアイコン or「サンプルを見る」リンクをクリック
2. サンプル文面ファイル（PDF or 画像）をモーダルで表示
3. ストレージから署名付きURLで取得（IF-03相当）
```

## 表示仕様

- PDF：インラインPDFビューワー
- 画像：img表示
