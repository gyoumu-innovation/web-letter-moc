# 機能設計書：送付リストアップロード機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-014 |
| 機能名 | 送付リストアップロード機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 新規注文作成画面 Step5（文面・送付先） |

---

## 機能説明

テンプレートに沿って作成した送付先リストファイルをアップロードする。  
アップロード後、件数を画面に表示して確認できる。

## 処理フロー

```
1. 「ファイルを選択」ボタンでファイル選択
2. ファイルの形式・サイズバリデーション（フロントエンド）
3. アップロード（IF-04）
4. バックエンドでファイル内容のバリデーション
   ├─ エラーあり → エラー内容を表示（アップロード失敗）
   └─ エラーなし → 件数を画面に表示
5. 「×」ボタンでファイル削除・再アップロード可能
```

## バリデーション

| チェック | エラーメッセージ |
|---|---|
| 形式不正 | .xlsx または .csv ファイルを選択してください |
| サイズ超過（10MB） | ファイルサイズは10MB以内にしてください |
| 必須列不足 | 「{列名}」列が見つかりません |
| 空ファイル | データが1件もありません |

## 外部IF
