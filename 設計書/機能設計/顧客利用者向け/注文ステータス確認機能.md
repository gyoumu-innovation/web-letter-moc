# 機能設計書：注文ステータス確認機能（注文詳細）

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-011 |
| 機能名 | 注文ステータス確認機能 |
| 対象ユーザー | 顧客利用者 |
| 対応画面 | 注文詳細画面（注文詳細_統合.html） |

---

## 機能説明

注文一覧から選択した注文の詳細情報・現在のステータスを確認する。  
仕様情報（封筒・便箋・差出人・文面・同梱物）をアコーディオンで参照できる。  
差し戻し時は差し戻し理由を表示する。

---

## 表示項目

### 注文基本情報

| 項目名 | 表示内容 |
|---|---|
| 注文番号 | orders.order_no |
| ステータスバッジ | 現在のステータス（色付き） |
| 商品種別・通数 | 封筒＋便箋 × N通 |
| 注文日 | orders.order_date |
| 発送予定日 | orders.estimated_ship_date |
| 発送日 | orders.shipped_date（発送済み時のみ） |

### 差し戻し理由（差し戻し時のみ）

| 項目名 | 表示内容 |
|---|---|
| 差し戻し理由 | return_reasons.reason |

### 仕様情報（アコーディオン）

| セクション | 表示内容 |
|---|---|
| 封筒 | サイズ・色・送付方法・糊付け・オプション |
| 便箋 | 書き方・フォント・オプション |
| 差出人情報 | 会社名・役職・住所 |
| 文面 | 宛名有無・本文 |
| 同梱物 | 同梱物名 |

---

## 処理フロー

```
1. 注文一覧から「確認する」ボタンクリックで遷移
2. order_id を元に注文情報・仕様情報を取得
3. ステータスに応じた表示制御
   - 差し戻し：差し戻し理由パネルを表示
   - 発送済み：発送日を表示
4. アコーディオンの開閉
5. パンくずリスト「注文一覧」クリック → 注文一覧画面へ戻る
```

---

## アクセス制御

- ログインユーザーの `company_id` と注文の `company_id` が一致する場合のみ閲覧可
- 一致しない場合はエラー画面（403）へ遷移
