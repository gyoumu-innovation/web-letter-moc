# 機能設計書：文面パターン機能（一覧・登録・編集・削除）

## 概要

| 機能ID | 機能名 | 画面 |
|---|---|---|
| F-U-021 | 文面パターン一覧機能 | template-list.html |
| F-U-022 | 文面パターン登録機能 | template-create.html |
| F-U-023 | 文面パターン編集・削除機能 | template-edit.html |

---

## F-U-021：文面パターン一覧

### 機能説明
ログインユーザーの企業に登録された文面パターンを一覧表示する。  
入稿フォームStep5でパターンを選択して本文に反映できる。

### 表示項目（カード形式）

| 項目 | 表示内容 |
|---|---|
| パターン名 | templates.name |
| 宛名設定 | has_recipient（あり/なし） |
| 本文プレビュー | body_text（先頭3行程度） |

### 操作

| ボタン | 動作 |
|---|---|
| 変更する | 編集画面へ遷移 |
| 削除する | 削除確認モーダル表示 |
| 新規作成 | 登録画面へ遷移 |

---

## F-U-022：文面パターン登録

### 入力項目

| 項目名 | 必須 | 説明 |
|---|---|---|
| パターン名 | ○ | 200文字以内。一覧での識別名 |
| 文頭に宛名を追加する | - | チェックボックス |
| 本文 | ○ | テキストエリア、文字数制限あり（要定義） |

### 処理フロー

```
1. 一覧画面「新規作成」→ 登録フォーム表示
2. 入力・バリデーション
3. templatesテーブルにINSERT
4. 一覧画面へ戻る
```

---

## F-U-023：文面パターン編集・削除

### 編集
登録と同じフォームに既存値を初期表示して更新。

### 削除
```
1. 一覧画面「削除する」クリック → 確認モーダル
2. 「削除する」確定 → 論理削除
```
※文面パターンは注文確定後にコピーが保存されるため、削除しても既存注文に影響なし。
