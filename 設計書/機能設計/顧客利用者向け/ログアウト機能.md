# 機能設計書：ログアウト機能

## 概要

| 項目 | 内容 |
|---|---|
| 機能ID | F-U-005 |
| 機能名 | ログアウト機能 |
| 対象ユーザー | 顧客利用者（ログイン済み） |
| 対応画面 | ログアウト確認画面（logout.html） |

---

## 処理フロー

```
1. ヘッダーのユーザーメニュー「ログアウト」から遷移
2. ログアウト確認画面を表示
3. 「ログアウトする」ボタンクリック
4. セッション破棄 / RememberMeトークン削除
5. ログイン画面へ遷移
```

## 関連テーブル

- セッションストア：セッション削除
- users：RememberMeトークン列をNULLに更新（該当する場合）
