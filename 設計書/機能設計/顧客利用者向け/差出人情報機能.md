# 機能設計書：差出人情報機能（一覧・登録・編集・削除）

## 概要

| 機能ID | 機能名 | 画面 |
|---|---|---|
| F-U-017 | 差出人情報一覧機能 | sender-info.html |
| F-U-018 | 差出人情報登録機能 | sender-info-create.html |
| F-U-019 | 差出人情報編集・削除機能 | sender-info-edit.html |

---

## F-U-017：差出人情報一覧

### 機能説明
ログインユーザーの企業に登録された差出人情報を一覧表示する。

### 表示項目（カード形式）

| 項目 | 表示内容 |
|---|---|
| 会社名・部署・役職 | sender_info.sender_company, department, position |
| 差出人名 | sender_info.name |
| 住所 | 郵便番号・都道府県・住所1・住所2 |

### 操作

| ボタン | 動作 |
|---|---|
| 変更する | 編集画面へ遷移 |
| 削除する | 削除確認モーダル表示 |
| 新規作成 | 登録画面へ遷移 |

---

## F-U-018：差出人情報登録

### 入力項目

| 項目名 | 必須 | 最大文字数 |
|---|---|---|
| 会社名 | - | 200文字 |
| 部署名 | - | 100文字 |
| 役職（肩書） | - | 100文字 |
| 差出人名 | ○ | 100文字 |
| 郵便番号 | ○ | 8文字（ハイフン含む） |
| 都道府県 | ○ | 10文字 |
| 市町村・番地 | ○ | 200文字 |
| 建物名など | - | 200文字 |

### 処理フロー

```
1. 一覧画面「新規作成」→ 登録フォーム表示
2. 入力・バリデーション
3. sender_infoテーブルにINSERT
4. 一覧画面へ戻る
```

---

## F-U-019：差出人情報編集・削除

### 編集
登録と同じフォームに既存値を初期表示して更新。

### 削除
```
1. 一覧画面「削除する」クリック → 確認モーダル表示
2. 「削除する」確定
3. 使用中チェック（注文で参照されていないか確認）
   ├─ 使用中 → エラーメッセージ（削除不可）
   └─ 未使用 → 論理削除
4. 一覧に反映
```
