# 帳票設計書

## 概要

本書はロボットレターWEB注文システムで出力・送付される帳票・帳票類似ファイルの仕様を定義する。

---

## 帳票一覧

| 帳票ID | 帳票名 | 出力形式 | 出力タイミング | 利用者 |
|---|---|---|---|---|
| R-01 | 注文確認メール | HTML/TXT メール | 注文確定時 | 顧客利用者 |
| R-02 | 差し戻し通知メール | HTML/TXT メール | 差戻し時 | 顧客利用者 |
| R-03 | 発送完了通知メール | HTML/TXT メール | 発送済み更新時 | 顧客利用者 |
| R-04 | 送付リストテンプレート | .xlsx | ダウンロード時（随時） | 顧客利用者 |
| R-05 | パスワード再設定メール | HTML/TXT メール | リセットリクエスト時 | 顧客利用者 |
| R-06 | 招待メール（初回パスワード設定） | HTML/TXT メール | 社内担当者がユーザー作成時 | 顧客利用者 |

---

## R-01：注文確認メール

### 概要
注文が確定（status: confirmed）になった際に顧客へ送信するメール。

### 送信条件
- orders.statusが'confirmed'に更新されたとき
- 送信先：orders.user_id → users.email

### 件名
```
【ロボットレター】ご注文を受け付けました（注文ID：{order_id}）
```

### 本文（テキスト形式）
```
{user_name} 様

この度はロボットレターをご利用いただきありがとうございます。
以下の内容でご注文を受け付けました。

■ 注文ID：{order_id}
■ 注文日時：{created_at}
■ 封筒サイズ：{envelope_size}
■ 書き方：{writing_direction}
■ フォント：{font_name}
■ 送付通数：{recipient_count}通

注文の進捗はWEB注文システムよりご確認いただけます。
{system_url}/order-list.html

ご不明な点がございましたら、担当者までお問い合わせください。

--
ロボットレター運営事務局
```

### パラメータ

| パラメータ | 取得元 |
|---|---|
| user_name | users.name |
| order_id | orders.id |
| created_at | orders.created_at |
| envelope_size | order_envelope_specs.size |
| writing_direction | order_stationery_specs.direction |
| font_name | order_stationery_specs.font |
| recipient_count | orders.recipient_count |
| system_url | 環境設定値 |

## R-04：送付リストテンプレート

### 概要
顧客が送付先を入力するためのExcelテンプレートファイル。

### ファイル形式
- 形式：.xlsx
- ファイル名：`送付リストテンプレート.xlsx`

### シート構成

**シート名：送付先リスト**

| 列名 | 必須 | 説明 | 最大文字数 |
|---|---|---|---|
| 郵便番号 | ○ | ハイフンあり（例：123-4567） | 8 |
| 都道府県 | ○ | 例：東京都 | 10 |
| 市町村・番地 | ○ | 例：渋谷区1-2-3 | 200 |
| 建物名など | - | マンション名・部屋番号 | 200 |
| 宛名（会社名） | - | 会社名 | 200 |
| 宛名（部署） | - | 部署名 | 100 |
| 宛名（氏名） | ○ | 個人名（宛名あり設定時に使用） | 100 |

**シート名：記入例**

1行目にサンプルデータを記載。注意事項をコメントで記入。

### 格納場所
- ストレージの固定パスに格納（バージョン管理）
- IF-03（署名付きURL）でダウンロード

---

## R-05：パスワード再設定メール

### 概要
パスワード再設定リクエスト時に送信するメール。

### 送信条件
- パスワード再設定フォームにメールアドレスを入力して送信したとき

### 件名
```
【ロボットレター】パスワード再設定のご案内
```

### 本文（テキスト形式）
```
ロボットレターをご利用いただきありがとうございます。

パスワード再設定のリクエストを受け付けました。
以下のURLよりパスワードを再設定してください。

{reset_url}

※このURLは{expire_minutes}分間有効です。
※お心当たりのない場合は、このメールを破棄してください。

--
ロボットレター運営事務局
```

### パラメータ

| パラメータ | 取得元 |
|---|---|
| reset_url | {system_url}/reset-password.html?token={token} |
| expire_minutes | 設定値（60分） |
| token | ランダム生成トークン（users.reset_token） |
